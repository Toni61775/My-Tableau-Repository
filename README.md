# 📊 Tableau Data Visualisation Project

## 📌 Overview
This project was completed as part of a **Data Technician Bootcamp**, focusing on developing practical skills in **Tableau** for data analysis and visualisation. The aim was to explore datasets (including retail/sales-style and analytical data) and transform raw data into **interactive, insightful dashboards** that support data-driven decision-making.

The project demonstrates the ability to **communicate insights through data storytelling** by building dynamic visualisations that highlight trends, patterns, and key business metrics.

---

## 🎯 Objectives
- Analyse real-world datasets to extract meaningful insights  
- Design and build **interactive dashboards** in Tableau  
- Apply visual storytelling techniques to present findings clearly  
- Use a variety of chart types to represent different data perspectives  

---

## 🛠️ Tools & Technologies
- **Tableau Public / Tableau Desktop**
- Excel / CSV datasets
- Data visualisation best practices

---
